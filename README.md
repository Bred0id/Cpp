# This is Phystech

Привет! 👋 Добро пожаловать в мой репозиторий.

Здесь я собираю материалы и решения по учебным активностям (в первую очередь ФПМИ/МФТИ).

## Описание содержимого веток

### 📌 C++ (базовый поток)
Ветка: **C++-base**  
Содержит полные решения всех контестов по C++ базового потока ФПМИ (кодставйл 1 семестр, 2025 год).  
Внутри ветки — навигация по задачам с кликабельными ссылками. Просто выбирайте нужную по названию и перед вами код решения.

---

## Замечание про “звёздочки”
Если в навигации задача помечена *, это **обязательная** задача (если она не зачтена, оценку выше **УД3** не поставят, в нашем потоке вообще оценку без них не выставляли).
